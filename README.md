pairs-web
=========

Website (Client) for PAIRS

Applying Bootstrap Theme from http://www.bootstrapzero.com/bootstrap-template/bolt

Dependency
---------
We are mainly using Bootstrap 3.2(http://getbootstrap.com/) as template.
